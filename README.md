🛠️ Komponen Rangkaian
1x Arduino Uno R3 (Otak sistem)
1x Breadboard
12x LED (4 Merah, 4 Kuning, 4 Hijau)
12x Resistor 220 Ohm (Biar LED nggak meledak!)
Kabel Jumper secukupnya

🚦 Konfigurasi Pin GPIO (Versi Iven)Sistem ini menggunakan pin digital Arduino Uno dengan pemetaan jalur sebagai berikut:Jalur Utara ⬆️: Merah (Pin 4), Kuning (Pin 3), Hijau (Pin 2).Jalur Timur ➡️: Merah (Pin 7), Kuning (Pin 6), Hijau (Pin 5).Jalur Selatan ⬇️: Merah (Pin 10), Kuning (Pin 9), Hijau (Pin 8).Jalur Barat ⬅️: Merah (Pin 13), Kuning (Pin 12), Hijau (Pin 11).

🛠️ Detail Teknis RangkaianMikrokontroler: Menggunakan Arduino Uno R3 sebagai pusat kendali.Komponen: Menggunakan total 12 LED dan 12 Resistor 220 Ohm untuk membatasi arus.Logika Program: Seluruh pin dikonfigurasi sebagai OUTPUT melalui fungsi pinMode().Urutan Jalan: Program dirancang berjalan searah jarum jam mulai dari Utara, Timur, Selatan, hingga Barat.
